# Auto_Python
Uma série de automações python para geração automatica de relatorios para uso via Excel/Power BI, utilizando as livrarias Pandas, Sql Alchemy entre outras.
Essas automações consistem de uma série de consultas SQL que recolhem informações triviais como por exemplo os numeros de WhatsApp dos clientes (com Ranking de melhor numero e número com o contato mais recente),
código de barras e linhas digitáveis do acordo dos clientes, para disparos massivos, analíticos de disparos massivos, contendo informações como status do envio, faixa de atraso do cliente e etc. 
São um total de 21 automações com diferentes formatos e funções.
